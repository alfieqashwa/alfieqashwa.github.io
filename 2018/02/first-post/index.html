Tutorial Kafe Part 1

 Jun 6, 2018
Tutorial Kafe Part 1 Pendahuluan Posting ini diperuntukkan untuk para user Kafe & Resto yang telah di training, khususnya back-office dan kasir. Tujuannya sebagai dokumentasi tutorial untuk membantu para user atau pengguna untuk memahami lebih baik tentang bagaimana cara mengoperasikan sistem aplikasi ini. Sebagai pendahuluan, sistem aplikasi ini adalah berbasis teknologi ERP yang dapat di implementasikan baik secara online (cloud-base) maupun offline (LAN). Sistem aplikasi ini berbasis web sehingga dapat dijalankan selama terinstall aplikasi browser seperti Chrome atau Firefox pada PC, Notebook, iPad, Tablet, atau SmartPhone.
